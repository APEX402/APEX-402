# APEX-402

APEX-402 (Agent Promotion Exchange Protocol) is the future AI-native Ad infrastructure for decentralized agent promotion, user interaction tracking and payment settlement.

## What is APEX-402

APEX-402 is a protocol based on x402, enabling AI agent to receive Ads content from MCP server, tracking user interactions after AI recommendation, and facilitating trustless ad payment settlement.

## Core Features

Beyond the central q402 protocol, APEX-402 introduces:

- /initiate-ad: generate a ad signature for interaction tracking.
- /complete-ad: verify the ad signature and settle ad payment.

## How to Deploy APEX-402

The APEX-402 facilitator can be easily deployed to the server via docker. See the Dockerfile for deployment instructions.
